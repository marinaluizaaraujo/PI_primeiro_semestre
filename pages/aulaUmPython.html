Introdução Python

Python é uma linguagem de programação flexível, poderosa e acessível, ideal tanto para quem está começando quanto para profissionais experientes, pois ela é fácil de ler e de escrever, muito usada em ciência de dados, web, automação, jogos e IA . Essa linguagem foi criada com o foco em legibilidade e simplicidade.
Nessa linguagem não é necessário o uso de ponto e virgula ( ; ) ou chaves ( {} ) para delimitar blocos,afinal a linguagem Python utiliza os espaços (tabulação) para organizar o código, fazendo o código ser executado linha por linha facilitando testes rápidos e prototipagem.
Um código Python pode rodar em vários sistemas sem grandes mudanças, funcionando em Windows, Linux, Mac, celulares e servidores.
